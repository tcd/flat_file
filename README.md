# FlatFile

Convenience methods for reading CSV, TSV, & JSON files.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'flat_file'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install flat_file
